# xbox_gamepass_subscriptions_sales
ENG: This spreadsheet analyzes Xbox Game Pass sales performance, with a specific focus on Minecraft and EA Play subscriptions.
The goal is to provide a clear, visual, and data-driven view of how each product contributes to total revenue.
POR: Esta planilha analisa o desempenho de vendas do Xbox Game Pass, com foco específico nas assinaturas do Minecraft e do EA Play.
O objetivo é fornecer uma visão clara, visual e orientada por dados sobre como cada produto contribui para o faturamento total.
